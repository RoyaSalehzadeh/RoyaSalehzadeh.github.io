Trust in Human–Robot Interaction for Law Enforcement Communication

This interdisciplinary NSF-funded project investigated how trust is established and calibrated during human–robot communication in high-stakes law enforcement settings. The team designed and deployed a teleoperated communication robot equipped with multimodal interaction capabilities—including audio, video, and display interfaces—to support remote officer–public communication. Contributions included robot development, field deployment, and analysis of officer feedback to understand how design and communication modalities influence trust. The project highlighted the importance of interaction design, transparency, and context in shaping trust in robotic systems used for public safety applications.

Purposeful Communication Framework for Human–Robot Collaboration in Manufacturing

This project examined communication as a central mechanism for improving safety, transparency, and coordination in human–robot collaboration (HRC) within modern manufacturing environments. Through a systematic review of contemporary HRC approaches, the team identified major sources of uncertainty in collaborative tasks—ranging from limited robot feedback to ambiguous human intent—and analyzed how different communication modalities influence trust and team performance. The project resulted in a unified framework for purposeful communication that highlights how structured, bidirectional exchanges between humans and robots can enhance situational awareness, support human-in-the-loop decision-making, and enable more reliable collaborative manufacturing systems.

EEG-Based Decoding of Finger Numeral Configurations

This project investigated how electroencephalography (EEG) signals can be used to decode fine grained human intention for adaptive human–robot interfaces. Machine-learning methods were developed to classify neural patterns associated with finger numeral configurations and event-related potential (ERP) scalp distributions. The work demonstrated that subtle neural signatures related to numerical cognition can be distinguished using supervised learning models, establishing a foundation for intention-driven robotic control and future neuroadaptive systems.

Closed-Loop EEG Integration for Rehabilitation Robotics

Building on EEG intention-recognition research, this project extended neural decoding into closed loop control for rehabilitation applications. Working with collaborators, EEG signals were integrated with a functional electrical stimulation (FES) cycling system to modulate cadence in real time based on user intent. The project explored how motor imagery and neural activity can guide adaptive rehabilitation devices and highlighted the need for robust signal-processing pipelines and adaptive learning strategies to support reliable real-time neuro-controlled movement.

BCI for Exploring Human Responses to Robotic Behaviors

This project examined how humans perceive and respond to varying robotic behaviors during collaborative tasks using brain–computer interface (BCI) methods. Robot expressivity, gesture patterns, and displays of vulnerability were manipulated while collecting EEG and behavioral measures to assess how these design features influence perceived safety, trust, and engagement. Neural and emotional responses were analyzed to identify patterns associated with different robot behaviors, demonstrating the utility of BCIs as continuous, objective indicators of user state. This work supports the development of adaptive robotic systems capable of adjusting communication strategies dynamically based on human feedback.
