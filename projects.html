Trust in Human–Robot Interaction for Law Enforcement Communication

This NSF-funded project studied how trust is established during human–robot communication in law enforcement scenarios. A teleoperated communication robot with multimodal interfaces (audio, video, and display) was developed to support remote officer–public interaction. The work examined how robot design and communication modalities influence trust in public safety contexts.

Purposeful Communication Framework for Human–Robot Collaboration in Manufacturing

This project studied communication as a key mechanism for improving safety and coordination in human–robot collaboration (HRC) in manufacturing. Through a systematic review of HRC systems, the work identified sources of uncertainty in collaborative tasks and analyzed how communication modalities influence trust and team performance. The project proposed a framework for purposeful human–robot communication to support more reliable and transparent collaboration.

EEG-Based Decoding of Finger Numeral Configurations

This project explored how electroencephalography (EEG) signals can be used to decode human intention for adaptive human–robot interfaces. Machine learning models were developed to classify neural patterns associated with finger numeral configurations and event-related potentials (ERPs), demonstrating that neural signatures of numerical cognition can be reliably distinguished for intention-driven robotic control.

Closed-Loop EEG Integration for Rehabilitation Robotics

This project extended EEG-based intention decoding to closed-loop control for rehabilitation. Neural signals were integrated with a functional electrical stimulation (FES) cycling system to modulate cadence in real time based on user intent, demonstrating how brain signals can guide adaptive rehabilitation devices.

BCI for Exploring Human Responses to Robotic Behaviors

This project studied how humans perceive and respond to different robotic behaviors during collaborative tasks using brain–computer interface (BCI) methods. EEG and behavioral data were collected while varying robot expressivity and gestures to understand how design features influence safety perception, trust, and engagement.
